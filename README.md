# Coffee Shop (Under Co.)
A coffee shop website demo.

[View Live](https://local-coffee-demo.netlify.app/)

## Technologies Used
- CSS3
- HTML5
- JavaScript

## Features
- Animated landing page (CSS keyframes)
- CSS grid
- Testimonials slider (JavaScript)
- Responsive design
